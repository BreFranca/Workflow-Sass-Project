## What's included

Within the download you'll find the following directories and files, logically grouping common assets and providing both compiled and minified variations. You'll see something like this:

```
framework/
├── Gruntfile.js
├── package.json
└── assets/
    ├── img/
    │   └── carousel-loader.gif
    ├── css/
    │   └── framework.css
    ├── fonts/
    │   ├── carousel.eot
    │   ├── carousel.svg
    │   ├── carousel.ttf
    │   └── carousel.woff
    ├── js/
    │   ├── framework.js
    │   ├── min/
    │   │    └── framework.min.js
    │   └── libs/
    │       ├── animate.js
    │       ├── carousel.js
    │       ├── gallery.js
    │       ├── jquery-3.1.0.min.js
    │       ├── mask.js
    │       ├── navs.js
    │       └── parallax.js
    └── sass/
        ├── framework.sass
        └── components/
            ├── _accordions.scss
            ├── _animate.scss
            ├── _carousel.scss
            ├── _classes.scss
            ├── _extends.scss
            ├── _flexBox.scss
            ├── _fonts.scss
            ├── _gallery.scss
            ├── _grid.scss
            ├── _gridFlex.scss
            ├── _gridFloat.scss
            ├── _mixins.scss
            ├── _modal.scss
            ├── _nav.scss
            ├── _normalize.scss
            ├── _padding.scss
            ├── _reset.scss
            └── _variables.scss

```
